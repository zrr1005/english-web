# SpeakEasy

用你喜欢的内容练英语口语——导入字幕、文本或 B站视频，然后听、写、读。

[English](README.md)

## 功能

- **多种导入方式**：粘贴文本、上传 SRT/VTT 字幕文件、或粘贴 B站链接自动抓取字幕
- **全篇听力**：默认展示全文，支持 TTS 整篇播放并以句级高亮。可切换「盲听模式」隐藏原文，进度条显示语音纹波形
- **译文对照**：原文下方展示中文翻译——B站字幕自带 / LLM 翻译 / 手动粘贴，三种来源
- **逐句听写**：逐句播放、默写、提交，逐词高亮反馈（正确/错误/遗漏）
- **影子跟读**：整篇自动播放+自动录音循环，可调速、可选手动/自动模式
- **进度追踪**：素材列表展示每个素材的最近听写分数和跟读分数
- **AI 增强**：可选配置 LLM API Key，解锁 B站中文翻译 + 写/读错误解释功能
- **离线可用**：全部数据存 IndexedDB，无需后端

## 快速开始

```bash
npm install
npm run dev
```

用 **Chrome** 打开（语音识别需要 Chrome 或 Edge）。

## 使用流程

```
导入内容 → 素材列表（含分数） → 全篇听力
                                  ├─ 盲听 + 语音纹
                                  ├─ 译文对照
                                  ├─ 逐句听写
                                  └─ 影子跟读
```

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS（Warm Academic 暖纸学术风）
- Web Speech API（语音识别 + 语音合成）
- Dexie.js（IndexedDB）
- OpenAI 兼容 LLM（用户自填 Key）
